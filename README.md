# 🎮 MojoLauncher (MJLauncher)

> **MojoLauncher** (también conocido como **MJLauncher**) es un fork/port de **PojavLauncher** que permite emular y jugar **Minecraft: Java Edition** en dispositivos móviles (Android) usando las librerías **LWJGL**.

Esta es una versión oficial extraída de **Google Play** (v14) y portada para uso libre.

## ✨ Características

- ✅ Emulación completa de Minecraft: Java Edition
- ✅ Basado en el código de PojavLauncher
- ✅ Soporte para mods, shaders y resource packs
- ✅ Controles táctiles personalizables
- ✅ Compatible con cuentas Mojang/Microsoft

## 📋 Especificaciones

| Campo | Valor |
|-------|-------|
| **Nombre** | MojoLauncher / MJLauncher |
| **Paquete** | `com.motorola.launcher3` |
| **Origen** | Google Play Store |
| **Base** | Fork de PojavLauncher |
| **Función** | Emular Minecraft: Java Edition en Android (LWJGL) |
| **Tamaño** | ~39 MB |
| **Versión** | 14 (API 34) |
| **Compatibilidad** | Android 12+ |

## 🚀 Instalación

### Método 1: ADB (Recomendado)
```bash
adb install MotoLauncher3.apk
```

### Método 2: Manual
1. Copia `MotoLauncher3.apk` al teléfono
2. Ábrelo desde el explorador de archivos
3. Activa "Fuentes desconocidas" si es necesario
4. Instala

## 🎯 Como usar

1. Instala el APK
2. Abre **MojoLauncher**
3. Inicia sesión con tu cuenta de Minecraft
4. ¡Juega a Java Edition en tu móvil!

## 📝 Notas

- ⚠️ Requiere permisos elevados (launcher de sistema)
- 🔧 Usa LWJGL para emular el motor de Java Edition
- 📱 Funciona en la mayoría de dispositivos Android 12+
