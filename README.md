# Moto Launcher 3 Port

APK extraído directamente de un dispositivo Motorola (Launcher3QuickStep).

## Información
- **Paquete:** `com.motorola.launcher3`
- **Origen:** `/system_ext/priv-app/Launcher3QuickStep/Launcher3QuickStep.apk`
- **Tamaño:** ~39 MB
- **Versión:** 26.1.2 (compatible con Android 14+)

## Instalación
```bash
# Desde ADB
adb install MotoLauncher3.apk

# O manualmente: copiar al teléfono y instalar desde el explorador
```

## Notas
- Es un launcher de sistema (priv-app), requiere permisos de sistema para reemplazar el launcher por defecto
- Funciona en la mayoría de dispositivos Android 12+
- Para usarlo como launcher predeterminado: Ajustes > Aplicaciones > Launcher por defecto
