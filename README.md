**MojoLauncher** (también conocido como **MJLauncher**) es un fork/port de **PojavLauncher**. Ambas aplicaciones permiten emular y jugar a **Minecraft: Java Edition** en dispositivos móviles (Android) usando las librerías **LWJGL**.

Launcher oficial de **Google Play** (versión 14), extraído y portado para uso libre.

## Información

| Campo | Valor |
|-------|-------|
| **Nombre** | MojoLauncher / MJLauncher |
| **Paquete** | `com.motorola.launcher3` |
| **Origen** | Google Play Store (luego extraído del sistema) |
| **Base** | Fork de PojavLauncher |
| **Función** | Emular Minecraft: Java Edition en Android (LWJGL) |
| **Tamaño** | ~39 MB |
| **Versión** | 14 (API 34) |
| **Compatibilidad** | Android 12+ |

## Instalar

```bash
adb install MotoLauncher3.apk
```

| Método | Comando |
|--------|---------|
| ADB | `adb install MotoLauncher3.apk` |
| Manual | Copiar APK al teléfono e instalar desde el explorador |

## Como usar

1. Instala el APK
2. Abre **MojoLauncher**
3. Inicia sesión con tu cuenta de Minecraft
4. Juega a Java Edition en tu móvil

## Notas

- Requiere permisos elevados (launcher de sistema)
- Usa LWJGL para emular el motor de Java Edition
- Funciona en la mayoría de Android 12+
